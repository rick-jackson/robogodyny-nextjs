# РобоГодини • Next.js версія

Сучасний веб-застосунок для обліку робочого часу з календарем, модальними формами, розрахунком заробітку (з урахуванням понаднормових та вихідних) та панеллю адміністратора.

## Особливості

- 📅 Інтерактивний календар з кольоровою індикацією завантаженості
- ⏱️ Точний розрахунок годин та заробітку (понаднормові, вихідні ×1.5 / ×2)
- ✈️ Підтримка відряджень з доїздом та статусом водія
- 👥 Перемикання між режимом співробітника та адміністратора
- 💾 Збереження даних у localStorage
- 🎨 Сучасний темний дизайн (Tailwind + кастомні стилі)

## Технології

- **Next.js 15** (App Router, Turbopack)
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (іконки)
- **date-fns** (робота з датами)

## Запуск проекту

```bash
cd robogodyny-nextjs
npm install
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000)

## Як користуватися

1. **Режим співробітника** — натискайте на дні календаря, щоб додавати/редагувати записи.
2. **Додати тестові дані** — кнопка внизу календаря.
3. **Адмін-режим** — перемикач у шапці. Оберіть співробітника, щоб переглянути його записи.
4. **Налаштування** — кнопка в шапці (зміна ставки тощо).
